隐私与安全

在币丰港，保护用户隐私和数据安全永远是第一要务。隐私和开放并不矛盾，我们对开放性的追求，不会以损害用户隐私为代价。

用户隐私
所有能够识别用户真实身份的数据将以高度加密的形式表示和存储。 在我们开放的数据中，用户信息只是一个匿名的ID，和比特币块链中的数据类似。
安全性
所有数据均采用AES-256算法进行加密。 用户密码将在浏览器端散列后传输，在服务器端还会利用Bcrypt算法进行二次散列。我们不接触原始密码，即使数据泄露，也不会扩大用户的损失。 系统所有数据都将定期备份，并妥善存储，避免数据丢失导致的事故发生。

Privacy and Security

At Coinport, we take user privacy and data security very seriously; and we believe privacy and transparency can co-exist beautifully.

User Privacy
All data that can be used to identify a user in the real world are encrypted and well protected. This ensures the anonymity of our open-data, similiar to cryptocurrency public ledgers.
Security
All data are encrypted using AES-256 before being persisted; passwords are hashed in the browser first then hashed again on server using Bcrypt algorithm. We keep multiple copies of all data and perform regular backups.
