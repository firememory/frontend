Privacy and Security

At Coinport, we take user privacy and data security very seriously; and we believe privacy and security can co-exist beautifully.

User Privacy
All data that can be used to identify a user in the real world are encrypted and well protected. This ensures the anonymity of our open-data, similiar to cryptocurrency public ledgers.
Security
All data are encrypted using AES before being persisted; passwords are hashed in the browser first then hashed again on server using Bcrypt algorithm. We keep multiple copy of all data and perform regular backups.
