# nodex
Frontend project of exchange using Node.js.

## build
### install Node.js
follow instructions in [http://nodejs.org](http://nodejs.org).

### set proxy of npm
```
npm config set registry=http://registry.npmjs.org
```

### install Express
```
sudo npm install express@3.4.8 -g
```

## run
```
cd main
sudo npm install
node app
```
